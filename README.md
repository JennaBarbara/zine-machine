This is a tool to help users create classic 8 page zines on letter-sized paper! Users can upload 8 images, reorder them if necessary and generate a correctly formatted 1 page PDF for printing.

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).
